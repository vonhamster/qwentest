Yii2 TypeScript Analog Framework

A modern web framework inspired by Yii2, built with TypeScript.

Key Features

  • ActiveRecord ORM: Easy database operations with SQLite
  • MVC Architecture: Clean separation of concerns
  • REST API: Full CRUD support for resources
  • Authentication: JWT-based auth with local database
  • View Engine: Template-based HTML rendering
  • Bootstrap 5: Modern, responsive UI components
  • Tauri Ready: Optimized for mobile and desktop apps

Technology Stack

  • TypeScript - Type-safe JavaScript
  • Express.js - Fast web server
  • better-sqlite3 - High-performance SQLite
  • JWT - Secure authentication tokens
  • Bootstrap 5 - Responsive CSS framework
